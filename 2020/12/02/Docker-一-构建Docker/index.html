三、vagrant相关命令

命令 含义
vagrant init 初始化
vagrant up 启动虚拟机
vagrant halt 关闭虚拟机
vagrant reload 重启虚拟机
vagrant ssh SSH至虚拟机
vagrant suspend 挂起虚拟机
vagrant resume 唤醒虚拟机
vagrant status 查看虚拟机运行状态
vagrant destroy 销毁当前虚拟机
vagrant box list 查看本地box列表
vagrant box add 添加box到列表
vagrant box remove 从box列表移除
